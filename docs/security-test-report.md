```
┌─────────────────────────────────────────────┐

                    ↓

│     - 格式验证 (32-64位十六进制)              │
```

## 已知限制

1. **设备指纹唯一性**: 理论上可能存在指纹冲突（极低概率）
2. **隐私模式**: 浏览器隐私模式可能导致指纹不稳定
3. **虚拟机/模拟器**: 可能绕过指纹验证（需要配合其他检测手段）
