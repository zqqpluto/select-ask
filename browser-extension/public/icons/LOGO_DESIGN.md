# Select Ask Logo 设计规范

## 设计理念
- **S** 和 **A** 字母组合，代表 "Select Ask"
- 白色 "S" + 浅蓝色 "A" 形成视觉层次
- 三色垂直渐变背景，营造深度和立体感
- 顶部高光效果增加光泽感
- 圆角设计（112px）柔和现代

## 颜色规范（2026-04-08 优化版）
- 渐变顶部：`#3B82F6` (Blue 500) - 明亮起始
- 渐变中部：`#2563EB` (Blue 600) - 主体色调
- 渐变底部：`#1E40AF` (Blue 800) - 深度收尾
- "A" 字母色：`#BFDBFE` (Blue 200) - 柔和对比
- 文字白色：`#FFFFFF`

## 字体
- Inter (Google Fonts)
- 字重：800 (Extra Bold)
- 字号：200px (512px 画布)

## 视觉效果
- 顶部高光：15% 透明度渐变，增强立体感
- 投影效果：y=4, blur=12, 37.5% 不透明度
- 圆角半径：112px (约 22% 边长)

## 导出尺寸
- icon.svg: 矢量源文件 (512x512)
- icon16.png: 16x16
- icon48.png: 48x48
- icon128.png: 128x128

## 使用场景
- Chrome 扩展图标
- 工具栏按钮
- 设置页面 favicon
- Chrome Web Store
