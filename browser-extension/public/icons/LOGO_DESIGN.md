# Select Ask Logo 设计规范

## 设计理念

### 当前版本：SA 融合艺术字 (2026-04-08)
- **S** 和 **A** 字母融合设计，形成独特的 monogram 风格
- S 的尾部自然流动连接到 A，创造视觉连贯性
- 艺术化处理的字母路径，增强品牌识别度
- A 字母包含镂空设计（counter），增加层次感

### 历史版本
- **版本 1**: 白色 "S" + 浅蓝色 "A" 分离式设计
- **版本 2**: 三色垂直渐变背景优化版
- **版本 3**: SA 融合艺术字设计（当前）

## 颜色规范

### 背景渐变（垂直）
- 渐变顶部：`#3B82F6` (Blue 500) - 明亮起始
- 渐变中部：`#2563EB` (Blue 600) - 主体色调
- 渐变底部：`#1E40AF` (Blue 800) - 深度收尾

### SA 字母渐变（对角线）
- 起始色：`#FFFFFF` (白色)
- 结束色：`#BFDBFE` (Blue 200)
- 渐变方向：左上到右下

### 辅助色
- A 镂空色：`#1E40AF` (与背景底部相同)
- 文字白色：`#FFFFFF`

## 字体/路径
- 基础字体：Inter (Google Fonts)
- 字重：800 (Extra Bold)
- 当前版本：自定义路径（非字体）

## 视觉效果
- 顶部高光：20% 透明度渐变，增强立体感
- 投影效果：y=4, blur=12, 37.5% 不透明度
- 圆角半径：112px (约 22% 边长)

## 导出尺寸
- icon.svg: 矢量源文件 (512x512)
- icon16.png: 16x16
- icon48.png: 48x48
- icon128.png: 128x128

## 使用场景
- Chrome 扩展图标
- 工具栏按钮
- 设置页面 favicon
- Chrome Web Store
