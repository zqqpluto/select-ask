# Select Ask Logo 设计规范

## 设计理念
- **S** 和 **A** 字母组合，代表 "Select Ask"
- 白色 "S" + 浅蓝色 "A" 形成视觉层次
- 深蓝色渐变背景传达专业、信任感
- 圆角设计（112px）柔和现代

## 颜色规范
- 主色：`#2563EB` (Blue 600)
- 深色：`#1E40AF` (Blue 800)
- 强调色：`#93C5FD` (Blue 300)
- 文字白：`#FFFFFF`

## 字体
- Inter (Google Fonts)
- 字重：800 (Extra Bold)

## 导出尺寸
- icon.svg: 矢量源文件 (512x512)
- icon16.png: 16x16
- icon48.png: 48x48
- icon128.png: 128x128

## 使用场景
- Chrome 扩展图标
- 工具栏按钮
- 设置页面 favicon
